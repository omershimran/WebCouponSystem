available actions

*create coupon

creates a new coupon in the database

*remove coupon

removes a coupon from the database

*update coupon

update the details of a specific coupon

*get coupon

view the details of a specific coupon, choosen by id

*get all coupons

view a list of all the coupons you created

*get all coupons by type

view a list of all the coupons you created of a certain type

*get all coupons by price

view a list of all the coupons you created up to a certain price

*get all coupons by date

view a list of all the coupons you created up to a certain end date

***to logout simply click the picture on the top right
